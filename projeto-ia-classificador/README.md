# 🤖 Classificador de Imagens com IA
 ## 📝 Descrição do Projeto
Este projeto utiliza **Machine Learning** (Aprendizado de Máquina) para identificar e classificar objetos/padrões em tempo real através da câmera ou upload de imagens. O modelo foi treinado utilizando a técnica de aprendizado supervisionado, onde forneci diversos exemplos de imagens para que a IA aprendesse a distinguir as classes com precisão.

Desenvolvido como um experimento prático para entender o funcionamento de redes neurais e visão computacional durante meus estudos de Engenharia de Software na **UNICID**.

 ## 🚀 Tecnologias Utilizadas
*   **Plataforma:** Teachable Machine (Google)
*   **Framework:** TensorFlow.js (para exportação e execução no navegador)
*   **Conceitos:** Visão Computacional, Classificação de Imagens e Redes Neurais.

 ## 📊 Resultados e Aprendizados
*   **Treinamento:** Aprendi a importância da diversidade do conjunto de dados (dataset) para evitar vieses no modelo.
*   **Acurácia:** O modelo demonstrou alta confiança na identificação das classes após o ajuste dos hiperparâmetros (épocas e taxa de aprendizado).
*   **Aplicação:** Entendi como modelos de IA podem ser integrados em aplicações web usando JavaScript.

 ## 🔧 Como Testar
1. Acesse o [link do modelo](https://teachablemachine.withgoogle.com/models/q2vRupbwG/).
2. Autorize o uso da câmera (se solicitado) ou faça upload de uma imagem.
3. Observe a barra de probabilidade classificar o objeto em tempo real.
