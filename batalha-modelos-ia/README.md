# ⚔️ Batalha de Modelos & Engenharia de Prompt (XML)
 ## 📝 Descrição do Projeto
Este projeto consistiu em um estudo comparativo de performance entre 7 grandes modelos de linguagem (LLMs), incluindo **Gemini, GPT, DeepSeek, Qwen, Grok, Maritaca e Claude**.

O objetivo foi testar a capacidade dessas IAs em processar um **Prompt Estruturado em XML** para gerar uma página Single Page (HTML5/CSS3) para a cafeteria fictícia "Grão & Pausa".

---
### 🛠 Estrutura do Prompt Utilizado
Utilizei uma estrutura XML para garantir precisão técnica:
- **Objetivo:** Criar página HTML/CSS responsiva.
- **Diretrizes de Design:** Paleta de cores específica (#ffffff, #1a1a1a, #bc6c25).
- **Obrigatoriedades:** Menu funcional, galeria de portfólio e transições suaves.

---
 ## 📊 Quadro de Análise Comparativa
Realizei uma análise baseada em critérios técnicos. Alguns destaques:
*   **Melhor Compreensão:** O **Gemini** demonstrou a melhor interpretação da estrutura XML[cite: 1].
*   **Prototipagem Rápida:** Escolhi o **Gemini** pela eficiência e inovação visual[cite: 1].
*   **Códigos Complexos:** O **DeepSeek** foi o escolhido para estruturas mais densas[cite: 1].
*   **Variação de Tokens:** A verbosidade variou drasticamente, de 650 (Maritaca) a 5.200 tokens (Claude)[cite: 1].

 ## 🚀 Tecnologias e Ferramentas
*   **Linguagens:** HTML5, CSS3.
*   **Engenharia de Prompt:** Estruturação XML.
*   **IAs Testadas:** OpenAI GPT, Google Gemini, Anthropic Claude, DeepSeek, Qwen, xAI Grok e Maritaca.
