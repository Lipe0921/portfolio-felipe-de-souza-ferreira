### 🏗 Estrutura do Repositório
Abaixo, a organização deste repositório para facilitar a navegação:

- `root/`: Arquivo de apresentação principal (este README) e scripts soltos.
- `/batalha-modelos-ia/`: Documentação e análise comparativa de modelos de IA.
- `/sql-prompt-engineer/`: Template de prompt XML para geração segura de queries SQL.

- `/Juros compostos/`: Códigos da calculadora financeira e simulador de taxas.
- `/Encurtador/`: Código-fonte e configurações da API de encurtamento de links.
- `/Gerador de Senhas Seguras/`: Interface e script do gerador de senhas seguras.
