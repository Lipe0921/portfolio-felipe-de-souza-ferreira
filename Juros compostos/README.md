# 📈 Simulador de Juros Compostos

Uma calculadora interativa desenvolvida para simular a evolução de investimentos ou o crescimento de dívidas com base em taxas de juros compostos.

## 🚀 Funcionalidades
* Cálculo dinâmico do valor total acumulado.
* Separação visual entre o valor inicialmente investido e os juros calculados.
* Interface responsiva para uso em computadores ou dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript (Lógica de cálculo e manipulação do DOM)

## 🔧 Como Executar
1. Baixe ou clone os arquivos deste repositório.
2. Abra o arquivo `index.html` diretamente em qualquer navegador web.
3. Insira o capital inicial, a taxa de juros e o tempo em meses para ver o resultado instantaneamente.
