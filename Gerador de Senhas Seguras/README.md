# 🔐 Gerador de Senhas Seguras

Ferramenta web para geração de senhas aleatórias e fortes, permitindo customizar o tamanho e os tipos de caracteres incluídos.

## 🚀 Funcionalidades
* Escolha do comprimento da senha (quantidade de caracteres).
* Opções para incluir ou excluir letras maiúsculas, números e símbolos.
* Sistema de cópia rápida para a área de transferência (Clipboard).

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript Vanilla

## 🔧 Como Executar
1. Salve os arquivos em uma pasta no seu computador.
2. Dê um duplo clique no arquivo `index.html` para abrir no navegador.
3. Configure as opções desejadas e clique em "Gerar Senha".
