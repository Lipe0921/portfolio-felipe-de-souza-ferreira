# 🗄️ SQL Prompt Engineer
 ## 📝 Descrição do Projeto
Este projeto aplica técnicas de **Engenharia de Prompt estruturado em XML** para automatizar a geração de consultas SQL. O foco é permitir que usuários obtenham dados de um banco acadêmico usando linguagem simples.

A escolha da estrutura XML baseia-se na minha análise anterior (Experiência 3), onde o modelo **Gemini** demonstrou maior precisão ao processar instruções delimitadas por tags[cite: 1].

 ## 🚀 Tecnologias Utilizadas
*   **Lógica de Banco de Dados:** SQL estruturado.
*   **Engenharia de Prompt:** XML Avançado (Instruções de contexto e segurança)[cite: 1].
*   **Modelo Recomendado:** Google Gemini (pela fidelidade aos itens do prompt[cite: 1]).

 ## 📊 Aprendizados Aplicados
*   **Segurança de Dados:** Implementação de travas no prompt para impedir comandos de deleção.
*   **Precisão Técnica:** Redução de bugs visuais e de sintaxe através de diretrizes claras[cite: 1].
